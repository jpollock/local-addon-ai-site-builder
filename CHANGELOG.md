# Changelog

All notable changes to the AI Site Builder Local addon will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- CI/CD pipeline with GitHub Actions
- Pre-commit hooks for code quality enforcement
- Automated testing and coverage reporting
- Build optimization and verification
- Release automation workflow

## [0.1.0] - 2025-11-29

### Added

- Multi-AI provider support (Claude, OpenAI, Gemini)
- AI-powered WordPress site scaffolding
- Site Information collection interface
- Navigation structure generator
- Content strategy and SEO planning
- Design system configuration
- Responsive layout settings
- Integration with Local by WP Engine
- Secure API key storage using keytar

### Changed

- Improved layout consistency using CSS Grid
- Enhanced section width consistency

### Fixed

- Navigation and Design section width issues
- Content width consistency across all sections

## [0.0.1] - 2025-11-22

### Added

- Initial project setup
- Basic addon structure for Local
- TypeScript configuration
- Jest testing framework
- Build scripts and entry point generation
