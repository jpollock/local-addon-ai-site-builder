# AI Site Builder for Local

AI-powered WordPress site scaffolding for [Local by WP Engine](https://localwp.com/).

## Features

- **Multi-AI Provider Support** - Claude, GPT-4, and Gemini
- **Intelligent Site Planning** - AI-powered content strategy
- **Figma Integration** - Design-aware site generation
- **Secure Storage** - API keys in system keychain
- **WordPress Integration** - Creates pages, menus, and content

## Installation

Download the correct package for your platform:

| Platform              | File                                      |
| --------------------- | ----------------------------------------- |
| macOS (Intel)         | `ai-site-builder-vX.X.X-darwin-x64.tgz`   |
| macOS (Apple Silicon) | `ai-site-builder-vX.X.X-darwin-arm64.tgz` |
| Windows               | `ai-site-builder-vX.X.X-win32-x64.tgz`    |
| Linux                 | `ai-site-builder-vX.X.X-linux-x64.tgz`    |

See [Installation Guide](INSTALL/) for detailed instructions.

## Requirements

- Local by WP Engine v6.0.0+
- API key from Claude, OpenAI, or Google Gemini
